# **App Name**: DevSphere Portfolio

## Core Features:

- Project Showcase: A dedicated section to display web projects, including titles, brief descriptions, technologies used (JavaScript, CSS, HTML, React, Next.js, TypeScript), and links to live demos or repositories.
- Skills Matrix: An interactive section that highlights the developer's proficiencies in languages (JavaScript, HTML, CSS, TypeScript), frameworks (React, Next.js), databases (Firebase, MongoDB, GraphQL), design tools (Figma), version control (Git), and methodologies (Responsive Design).
- About Me & Contact Form: A personalized 'About Me' section detailing Simon Enrique Ramirez Ferrer's background, professional interests, personal brand (Tekron Web Studio), nationality (Venezuelan), and language skills (native Spanish, basic English for code), accompanied by a contact form for inquiries from potential employers or collaborators.
- AI-Enhanced Project Briefing Tool: An AI tool that can generate concise and impactful project summaries based on user-provided keywords, technologies, and objectives, helping the developer quickly refine their project descriptions.
- Responsive Layout Engine: Implementation of responsive design principles ensuring the portfolio adapts flawlessly to various screen sizes and devices (desktops, tablets, mobile), showcasing proficiency in this area.

## Style Guidelines:

- Color scheme: Dark. Conveys professionalism and focus. Primary color (interactivity, accents): A deep, muted indigo blue (#262673) to represent technical depth and innovation. Background color: A very dark, subtle bluish-grey (#16161D) for an immersive canvas. Accent color (highlights, calls to action): A vibrant sky blue (#5299E0) that provides excellent contrast and energy.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and commanding presence. Body text font: 'Inter' (sans-serif) for high readability, neutrality, and clean aesthetics in project descriptions and skill listings. Code snippets: 'Source Code Pro' (monospace) for clear and legible code display.
- Utilize a minimalist icon set with clean lines, prioritizing SVG formats for scalability. Icons should visually represent technologies (e.g., React logo, JavaScript icon) or actions (e.g., link, contact) with a modern aesthetic.
- A grid-based layout for project showcase, allowing for easy scanning and a structured feel. Generous whitespace to enhance readability and give content room to breathe. Clear separation between sections with prominent headings.
- Subtle and purposeful animations for UI feedback, such as hover effects on project cards, smooth scrolling between sections, and clean transitions when navigating, avoiding anything overly distracting.
